# 💣 DEFCON — Bomb Disposal Unit

A thrilling browser-based bomb defusal game inspired by *Keep Talking and Nobody Explodes*.
Race against the clock to defuse a bomb by solving multiple interactive modules before time runs out!

---

## 🎮 Play Now

👉 **[Click here to play](https://pekalavinayak.github.io/Defcon/)**

---

## 🕹️ How to Play

1. Choose your **difficulty** — Easy, Medium, or Hard
2. A bomb will appear with multiple **modules** you need to solve
3. Solve all modules and enter the **disarm code** to defuse the bomb
4. If you make too many mistakes or run out of time — **BOOM** 💥

---

## 🧩 Game Modules

The bomb contains several different puzzle modules, each requiring a different skill:

| Module | Description |
|--------|-------------|
| 🔌 **Wire Cutting** | Cut the correct wire based on given clues |
| 🔢 **Keypad Code** | Enter the correct disarm code |
| 🔘 **Button Panel** | Press buttons in the right sequence |
| 📻 **Frequency Tuner** | Tune to the correct signal frequency |
| 🧠 **Memory Puzzle** | Remember and repeat sequences |
| ⚙️ **Simon Says** | Follow the color/pattern instructions |

## ⚙️ Difficulty Levels

| Level | Time | Strikes Allowed | Modules |
|-------|------|-----------------|---------|
| 🟢 **Easy (Standard)** | 5 mins | 3 | 3 modules |
| 🟡 **Medium (Advanced)** | 3 mins | 2 | 5 modules |
| 🔴 **Hard (Critical)** | 2 mins | 1 | 7 modules |

---

## 🏆 Scoring System

- ✅ Each module solved = **points earned**
- ⏱️ **Time Bonus** — faster defusal = more points
- ⭐ **Perfect Bonus** — no strikes = extra reward
- 📈 **Multiplier** increases with difficulty
- ❌ Mission failed = **50% score penalty**

## 📈 Progression

- Each difficulty has **20 levels** to complete
- Levels get harder as you progress
- Unlock **Prestige** by clearing all 20 levels in a difficulty
- Your progress and score are **saved automatically** in the browser
